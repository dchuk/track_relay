# track_relay

A Rails gem for unified, typed event tracking that fans out to multiple destinations (GA4, Ahoy, custom subscribers) from a single source-of-truth catalog, built on ActiveSupport::Notifications.

**Core value:** One catalog, many destinations: eliminate dual event vocabularies between marketing and product analytics in Rails apps.
